# Uplift Modeling Project

Проект посвящен uplift-моделированию для маркетинговой задачи Яндекс Еды. Цель: найти пользователей, которым действительно стоит отправлять промокод, чтобы максимизировать эффект от воздействия и не тратить бюджет на неэффективную рассылку.

## Что сделано

В основном ноутбуке [final_project_template.ipynb]() выполнены все этапы проекта:

- первичная подготовка данных и проверка пропусков;
- EDA с визуализациями и выводами;
- проверка статистической значимости различий между control и treatment;
- корреляционный анализ признаков;
- обучение трех uplift-моделей: `S-learner`, `T-learner`, `X-learner`;
- сравнение моделей по `Uplift AUC`, `Qini AUC`, `Uplift@30`;
- подбор гиперпараметров для финальной модели через `Optuna`;
- построение `Uplift Curve`, `Qini Curve` и графика uplift по перцентилям;
- реализация класса `UpliftModelInference` для инференса.

## Итоговая модель

В качестве финальной модели выбран `X-learner`.

Честная финальная оценка на test-выборке:

- `Uplift AUC: 0.0299`
- `Qini AUC: 0.0665`
- `Uplift@30: 0.0360`

Дополнительно был проведен эксперимент с генерацией новых признаков, но он не улучшил качество по `Uplift@30`, поэтому в финальную модель новые признаки не включались.

## Запуск проекта

Создать и активировать окружение:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

После этого можно запускать Jupyter Notebook и открывать основной файл:

```bash
jupyter notebook final_project_template.ipynb
```
